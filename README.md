# Copy Directory Structure

This PowerShell script copies the directory structure from a source folder to a target folder without copying the file contents. For Windows PCs

## Usage

1. **Save the Script**:
   - Save the `CopyDirectoryStructure.ps1` script to a location on your computer.

2. **Run the Script**:
   - Open PowerShell with administrative privileges.
   - Navigate to the directory where the script is saved.
   - Run the script by typing `.\CopyDirectoryStructure.ps1` and pressing `Enter`.

3. **Follow the Prompts**:
   - The script will prompt you to enter the source directory path.
   - The script will then prompt you to enter the target directory path.
   - The script will copy the directory structure from the source to the target without copying the file contents.

## Example

```powershell
PS C:\Scripts> .\CopyDirectoryStructure.ps1
Enter the source directory path: C:\Example\SourceFolder
Enter the target directory path: C:\Example\TargetFolder
Directory structure copied successfully from C:\Example\SourceFolder to C:\Example\TargetFolder

## Notes

    Ensure that you have the necessary permissions to read the source directory and write to the target directory.
    The script will create the target directory if it does not already exist.
